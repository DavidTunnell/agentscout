Anthropic API key

AgentScout uses your Anthropic API key to analyze captures and produce recommendations. The key is stored in the OS keychain (Windows Credential Manager / macOS Keychain / Linux Secret Service), never in a config file or sent anywhere except api.anthropic.com.
